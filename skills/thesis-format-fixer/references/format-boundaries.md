# Format Boundaries

- Only process `.docx` files.
- Do not process PDF, legacy `.doc`, or Word plugin workflows.
- Do not generate, rewrite, summarize, translate, or polish thesis content.
- Do not copy real thesis text into logs, prompts, reports, README files, or samples.
- Treat LLM output as advisory structure labels only.
- Let JSON rules and deterministic repair code decide all format changes.
- Always produce a manual-fix checklist for issues the engine cannot safely repair.
